==  Jade Editor behaviour


Events:
   jade:load  - load a named jade, render on the canvas
   jade:save  - save the current jade

The Input Text Area node (id="jade-texted") carries _shader / _shader_param* style
attrs; the engine reads them every frame and calls drawImageWithShader().
